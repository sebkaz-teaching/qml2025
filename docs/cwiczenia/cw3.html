    üßê Wyja≈õnienie Dzia≈Çania Funkcji Kosztu w VQA (Mini-Eksperyment)

    W tym prostym eksperymencie z Wariacyjnym Algorytmem Kwantowym (VQA), optymalizator (Gradient Descent) minimalizuje funkcjƒô kosztu, choƒá nie zosta≈Ça ona jawnie zdefiniowana jako np. Mean Squared Error (MSE). Dzieje siƒô tak, poniewa≈º w bibliotekach takich jak PennyLane, optymalizator domy≈õlnie przyjmuje, ≈ºe to, co zwraca \(\mathbf{QNode}\) jest funkcjƒÖ celu, kt√≥rƒÖ nale≈ºy minimalizowaƒá.

    1. Definicja Kosztu Przez Operator PauliZ

    Tw√≥j obw√≥d qn(theta) jest zdefiniowany nastƒôpujƒÖco:

    \[\mathbf{qn}(\theta) = \bra{\psi(\theta)} \mathbf{Z} \ket{\psi(\theta)} = \langle \mathbf{Z} \rangle\]

    Poniewa≈º \(\mathbf{qn}(\theta)\) zwraca Warto≈õƒá OczekiwanƒÖ operatora PauliZ (\(\langle \mathbf{Z} \rangle \in [-1, 1]\)), optymalizator traktuje ten wynik jako funkcjƒô kosztu \(\mathbf{C}(\theta)\), kt√≥rƒÖ nale≈ºy sprowadziƒá do najni≈ºszej mo≈ºliwej warto≈õci, czyli \(-1\).

    \[\text{Optymalizator dƒÖ≈ºy do minimalizacji: } \mathbf{C}(\theta) = \langle \mathbf{Z} \rangle\]

    2. Cel Obwodu Kwantowego

    W tym kontek≈õcie, obw√≥d VQA jest u≈ºywany do znalezienia stanu w≈Çasnego operatora \(\mathbf{Z}\) odpowiadajƒÖcego najni≈ºszej warto≈õci w≈Çasnej (\(\lambda = -1\)). Jest to analogiczne do algorytmu Variational Quantum Eigensolver (VQE), kt√≥ry s≈Çu≈ºy do znajdowania energii stanu podstawowego (najni≈ºszej warto≈õci w≈Çasnej Hamiltonianu).

    • Po≈ºƒÖdany Stan: Stan \(\ket{1}\), dla kt√≥rego \(\mathbf{Z}\ket{1} = -1\ket{1}\) (minimalna warto≈õƒá oczekiwana).
    • Architektura Obwodu:
      1. qml.X(wires=0): Ustawia kubit w stan \(\ket{1}\).
      2. qml.RY(theta, wires=0): Wprowadza parametr \(\theta\), kt√≥ry perturbuje stan \(\ket{1}\).
    • Dzia≈Çanie Optymalizatora: ZaczynajƒÖc z \(\theta\) bliskim zera (gdzie \(\langle \mathbf{Z} \rangle \approx -1\)), optymalizator bardzo szybko dƒÖ≈ºy do kƒÖta \(\theta\), kt√≥ry minimalizuje perturbacjƒô, czyli \(\theta \to 0\) lub \(\theta \to 2\pi\). W tych punktach stan wraca do \(\ket{1}\), a koszt osiƒÖga swoje minimum \(-1\).

    üí° Podsumowanie

    Eksperyment ten demonstruje podstawowƒÖ funkcjonalno≈õƒá wariacyjnego uczenia maszynowego w kwantowym kontek≈õcie, gdzie sam obw√≥d kwantowy mo≈ºe definiowaƒá funkcjƒô celu (Hamiltonian), a klasyczny optymalizator znajduje parametry \(\theta\), kt√≥re minimalizujƒÖ tƒô funkcjƒô. Jest to archetyp podej≈õcia hybrydowego (klasyczny optymalizator + kwantowe przetwarzanie).

    import pennylane as qml
import pennylane.numpy as np
import matplotlib.pyplot as plt

# 1. Definicja UrzƒÖdzenia i Obwodu Kwantowego
dev = qml.device('default.qubit', wires=1)

@qml.qnode(dev)
def qn(theta):
    qml.X(wires=0)       # Ustawia stan na |1> (expval Z = -1)
    qml.RY(theta, wires=0) # Obr√≥t wok√≥≈Ç Y (zmienia expval Z)
    return qml.expval(qml.PauliZ(0)) # Funkcja celu: minimalizujemy expval Z

# 2. Inicjalizacja Parametr√≥w i Optymalizatora
# Zaczynamy z theta, gdzie koszt nie jest minimalny, aby zobaczyƒá proces optymalizacji.
# Np. theta = pi/2, gdzie RY(pi/2) na |1> daje 1/sqrt(2)(|1> - |0>), a expval Z = 0.
theta_init = np.array([np.pi-0.01], requires_grad=True) 
opt = qml.GradientDescentOptimizer(stepsize=0.1)

epochs = 100

# Listy do przechowywania historii treningu
cost_history = []
theta_history = []

# 3. Pƒôtla Treningowa
theta = theta_init # Ustawienie poczƒÖtkowej warto≈õci theta

print("--- Rozpoczƒôcie treningu ---")
for epoch in range(epochs):
    # Wykonaj krok optymalizatora i oblicz koszt
    theta, cost = opt.step_and_cost(qn, theta)
    
    # Zapisz historiƒô
    cost_history.append(cost)
    theta_history.append(theta[0]) # theta jest tablicƒÖ numpy, bierzemy tylko warto≈õƒá

    if epoch % 10 == 0 or epoch == epochs - 1:
        print(f"Epoka: {epoch:3d} | KƒÖt theta: {theta.item():.4f} | Koszt: {cost.item():.4f}")

print("\n--- Trening zako≈Ñczony ‚úÖ ---")
print(f"Ko≈Ñcowy kƒÖt theta: {theta.item():.4f}") 
print(f"Ko≈Ñcowy koszt: {cost.item():.4f}")

# 4. Wizualizacja Historii Treningu
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# Wykres kosztu
axes[0].plot(range(epochs), cost_history, color='blue', linewidth=2)
axes[0].set_title('Ewolucja Funkcji Kosztu')

axes[0].grid(True, linestyle='--', alpha=0.6)

axes[0].legend()


# Wykres kƒÖta theta
axes[1].plot(range(epochs), theta_history, color='green', linewidth=2)
